Vuex的使用总结
发表于 2020-08-01| 更新于 2020-08-09
阅读量:

vuex是一个专门未vue.js应用程序开发的状态管理模式。它采用集中式存储管理应用的所有组件的状态，并以相应的规则保证状态以一种可预测的方式发生变化。

起步

使用 npm install vuex –save
新建store文件夹/index.js

在将store挂载到Vue的根组件上

main.js中

5个核心概念

store

其仓库

mutation

用于同步更新store中的数据

action

用于异步更新store中的数据

getter

相当于计算属性

modules

将store进行分块

其中modules下的user.js这么写

访问时使用this.$store.state.user.user可以得到3
使用this.$store.dispatch('asyncChnageUser')可以异步修改user模块中的user
使用this.$store.commit('changeUser')可以同步修改user模块中的user
使用this.getters.getUserAge 可以动态计算岁数

一次性引入所有modules

module中开启命名空间

在module中开启命名空间,加一个属性namespaced:true即可

进行action修改时

this.$store.dispatch('user/asyncChnageUser')

进行mutations修改时

this.$store.dispath('user/changeUser')

进行获取getters时

this.$store.getters['user/getUserAge']
