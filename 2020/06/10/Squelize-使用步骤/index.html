Squelize 使用步骤
发表于 2020-06-10| 更新于 2020-08-24
阅读量:

用于记录sequlize的使用

  • 初始化项目,生成package.json

  • 安装express

  • 进入项目

  • 安装sequelize

  • 安装mysql2

  • 安装sequelize-cli

  • 初始化项目目录

  • 目录介绍

    文件 作用
    config 数据配置
    Migration 迁移文件，用于新增数据库表等
    Models 模型
    Seeders 用于添加测试数据

  • 创建数据库

  • 建model和迁移文件

  • 运行迁移，生成数据表

  • 建种子文件

  • 在seeders文件夹下，运行迁移

  • 在routers中复制index.js修改为articel.js

  • app.js中操作

  • 使用nodemon热更新服务器
    1. 安装nidemon
    2. 修改package.json配置文件
      将
      script:{
      'dev':'node app.js'
      }
      修改为
      script:{
      'dev':'nodemon app.js'
      }
